# Quote Me

**Quotation Maker** is a professional web application for generating fire safety equipment quotations.

## Project Structure

- `quotation-maker/`: The Next.js web application source code.

## Getting Started

Navigate to the `quotation-maker` directory to run the application:

```bash
cd quotation-maker
npm run dev
```

For more details, see [quotation-maker/README.md](quotation-maker/README.md).
